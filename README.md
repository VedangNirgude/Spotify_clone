# Spotify Web Player Clone 

A responsive clone of Spotify's web player built using **HTML5** and **CSS3**. This project focuses on recreating Spotify's desktop interface while practicing modern CSS layouts, Flexbox, responsive design, and UI styling.

## Live Demo

https://yourusername.github.io/spotify-clone/

## Features

- Responsive Spotify-inspired UI
- Sidebar navigation
- Sticky top navigation bar
- Music player with playback controls
- Volume and progress sliders
- Album information section
- Playlist and podcast cards
- Hover animations and transitions
- Mobile responsive using media queries
- Font Awesome icons

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome
- Google Fonts (Montserrat)
